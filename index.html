Third Generation Programming (3GL)

Third generation programming started in the mid 1960's, it was so big because it was the first generation that was high leveled. Making the machine more independent by improving support for data types, and making programming expressions that favors the programmer and not the computer.

Birth of modern day programming

To pick up where we left off; 3GL or Third Generation Programming, was the first step toward programmmer friendly software.Things like Fortran, COBOL, & ALGOL are examples of early 3GL software. Later software developed would be Java, Python, C++, C#, etc. Most 3GL languages support structured programming.

Languages

Most popular general-purpose languages today, such as C, C++, C#, Java, BASIC and Pascal, are also third-generation languages, although each of these languages can be further subdivided into other categories based on other contemporary traits.
*Most 3GLs support structured programming

structured programming

Structured programming is a programming paradigm aimed at improving the clarity, quality, and development time of a computer program by making extensive use of the structured control flow constructs of selection (if/then/else) and repetition, block structures, and subroutines.

Examples Of 3GLs

Fortran

Fortran is a general-purpose, compiled imperative programming language that is especially suited to numeric computation and scientific computing.

COBOL

COBOL is a compiled English-like computer programming language designed for business use. It is an imperative, procedural and, since 2002, object-oriented language. COBOL is primarily used in business, finance, and administrative systems for companies and governments.
